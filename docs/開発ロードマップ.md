## 技術スタック

### 必須
- Minecraft Bedrock 1.21.90+
- Script API v2.0.0
- TypeScript 5.0+
- Node.js（ビルド環境）

### 開発ツール
- VS Code
- Minecraft Bedrock Dedicated Server（テスト用）
- Git（バージョン管理）

## リスクと対策

### 技術的リスク
- Script API仕様変更 → ベータ版は使用せず安定版のみ使用
- パフォーマンス問題 → 早期からプロファイリング実施
- マルチプレイ同期 → サーバー権威設計を徹底

### スケジュールリスク
- 複雑な魔法陣認識 → シンプルなアルゴリズムから段階的実装
- UI実装の困難さ → 基本UIから開始、段階的改善

## 成功指標

- 全システムが設計通り動作する
- シングル/マルチプレイ両対応
- 20人同時プレイで安定動作
- 魔法陣認識が1秒以内
- メモリ使用量が妥当な範囲内
