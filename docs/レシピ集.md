### 2.1.1 火属性加工魔法陣
| 入力アイテム | 出力アイテム | 備考 |
|-------------|-------------|------|
| ミスリル合金片 | ミスリルインゴット | 高温精錬効果 |
| ミスリル圧縮片 + 石炭 | ミスリル合金片 | 高温精錬効果 |
| 火の魔力結晶×9 | 火の濃縮魔力結晶 | 濃縮効果 |
| 火の濃縮魔力結晶×9 | 火の高純度魔力結晶 | 高純度化 |
| 火の高純度魔力結晶×9 | 火の究極魔力結晶 | 究極化 |
| アメジスト | 火の魔力結晶 | 魔力結晶化 |
